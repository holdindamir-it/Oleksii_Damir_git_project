# Mini-Projet : Gestion de projet avec Git et GitHub

Ce dépôt contient le code source d'un site web simple développé dans le cadre du module de Gestion de projet (Licence Informatique 2A - Université de Lyon 2).

## Objectif
L'objectif principal n'est pas la complexité du code, mais la démonstration de nos compétences à travailler en équipe en utilisant les outils de versionnement :
- Création de branches (feature-css, feature-content, etc.)
- Utilisation des Pull Requests pour la revue de code.
- Création de tags pour les versions finales (v1.0).
